# CodeAlpha Bioinformatics Internship - Task 1

## DNA/Protein Sequence Analysis Using BLAST

### Description
This project demonstrates DNA/Protein sequence analysis using the NCBI BLAST tool. The Human Insulin (INS) protein sequence was retrieved from the UniProt database and analyzed to identify homologous sequences. The BLAST results were interpreted based on sequence identity, query coverage, alignment score, and E-value.

### Tools Used
- UniProt
- NCBI BLASTP
- Microsoft Word
